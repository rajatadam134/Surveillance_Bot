The Hardware folder contains the Rough Circuit Diagram, Components Required , their Connections , etc

Build the Bot accordingly.

Now coming to the software part,
- Setup the raspberry PI
- Install all the required Libraries , MJPG file according to the file
- install the mobile application
- Run the Code in Raspberry PI
- you have to train your own ML-model. I have used teachable machine website to train the model. 
- I have provided you with a small Dataset of Fire Images which can work on small scale  projects
- you can train your own model for fire or face detection or any model you want
- i have provided a jupyter file in ML_savedmodel folder you can use that by copying it to where the model you have trained is stored (THIS FILE IS ONLY FOR FIRE DETCTION)
