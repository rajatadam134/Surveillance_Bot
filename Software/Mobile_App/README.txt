You can directly use the .apk file to install the application in you mobile phone 

OR

You can use the .aia file to edit the app on your own accord using the MIT app inventor




After the installation of the app, First ensure the Raspberry PI and the Mobile Phone are connected to the same network(WiFi)
Enter the IP address that the Raspberry PI is connected to 
- You can get it by using " hostname -I " in the terminal of raspberry PI
-then Press the Set IP button 
-The port is set to :5000 by default
